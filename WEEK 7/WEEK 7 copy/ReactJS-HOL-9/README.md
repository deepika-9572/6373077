# Cricket App

## Objective
A simple React app demonstrating ES6 features like:
- map()
- Arrow functions
- Destructuring
- Array merging

## Components
- **ListofPlayers**: Displays player names and scores, and filters low scorers.
- **IndianPlayers**: Displays odd/even team players and merged lists.

## To Run
```bash
npm install
npm start
```

## Prerequisites
- Node.js
- NPM
- Visual Studio Code
